How to Install & Use
  1. Add to Bookmarks: Drag the blue button above to your browser's bookmarks bar
  2. Go to LinkedIn: Navigate to LinkedIn.com and log in to your account
  3. Search for Jobs: Go to the jobs section and search for positions you're interested in
  4. Run the Bot: Click the bookmarklet in your bookmarks bar to start the automation
  5. Monitor Progress: Watch the progress modal and stop anytime if needed
What You'll See
  • Total Found: Number of job cards detected on the page
  • Eligible: Jobs with "Easy Apply" button available
  • Applied: Successfully submitted applications
  • Skipped: Jobs without Easy Apply or incomplete applications
  • Errors: Failed applications due to technical issues
Smart Detection

Automatically detects "Easy Apply" jobs and skips others to save time and avoid errors.

Real-time Progress

Live progress tracking with detailed counts and visual feedback.

Stop Anytime

Clean UI

Troubleshooting
  • No jobs found: Make sure you're on a LinkedIn job search results page
  • Applications failing: Check that your LinkedIn profile is complete
  • Bookmarklet not working: Try refreshing the page and running it again
  • Modal not appearing: Check if popup blockers are preventing the overlay
